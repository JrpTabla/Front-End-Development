HTML Images

An image can be shown using <img> tag along with its src attribute.

<img src="https://place-puppy.com/300x300">

<img src="./images/zuittlogo.png">

Valid image extension are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp
